Website statis sederhana yang dibuat menggunakan **HTML dan CSS** dengan layout profil dan tema **Neon Galaxy**.  
Proyek ini menampilkan halaman dengan header, navbar, gambar utama (hero), serta beberapa section profil dengan gambar dan deskripsi.

## ✨ Fitur
- Header dan navigation menu
- Hero image/banner utama
- Section profil dengan gambar kiri dan kanan
- Desain tema neon galaxy (ungu dan cyan)
- Layout responsif sederhana

## 🛠️ Teknologi
- HTML5
- CSS3
